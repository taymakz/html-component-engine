# HTML Component Engine

A lightweight Vite plugin for HTML components.

## Quick Start

```bash
npx html-component-engine
```

## Commands

```bash
npm run dev    # Development server
npm run build  # Production build
```

## Project Structure

```
src/
  index.html
  about.html
  components/
    Header.html
    Footer.html
    Card.html
```

## Build Output

```
dist/
  index.html
  about.html
```
